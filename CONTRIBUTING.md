# Guía de Contribución - Game Turbo Pro

Gracias por tu interés en contribuir a Game Turbo Pro.

## 🎯 Cómo Empezar

### 1. Fork el Repositorio
Haz clic en "Fork" en GitHub.

### 2. Clona tu Fork
```bash
git clone https://github.com/TU_USUARIO/Gameturbopro.git
cd Gameturbopro
```

### 3. Crea una Rama
```bash
git checkout -b feature/tu-feature-name
```

## 📋 Para Reportar Bugs

1. Verifica que no exista el bug
2. Abre un Issue con:
   - Título: `[BUG] Descripción corta`
   - SO, versión de Python
   - Pasos para reproducir
   - Logs si están disponibles

## 💡 Para Proponer Features

1. Abre una Discusión en GitHub
2. Describe tu idea y obtén feedback
3. Luego crea un Issue y desarrolla

## 💻 Para Enviar Código

1. Sigue la guía de estilo (PEP 8)
2. Escribe tests unitarios
3. Documenta tu código
4. Commit limpio y descriptivo
5. Abre un Pull Request
