# Guía de Inicio Rápido - Game Turbo Pro

## 🚀 5 Minutos para Empezar

### Paso 1: Clonar y Configurar (1 min)
```bash
git clone https://github.com/farru27/Gameturbopro.git
cd Gameturbopro
python setup.py
```

### Paso 2: Instalar Dependencias (2 min)
```bash
pip install -r requirements.txt
```

### Paso 3: Ejecutar Game Turbo Pro (30 seg)
```bash
python main.py
```

### Paso 4: Usar la Aplicación (1.5 min)
1. 📁 Haz clic en "EXAMINAR ARCHIVOS"
2. Selecciona el ejecutable de tu juego
3. Elige un perfil (⚡ Performance, ⚖️ Balanced, etc.)
4. 🚀 Haz clic en "LANZAR JUEGO"
5. 📊 Observa las estadísticas en tiempo real

## 🎮 Perfiles Recomendados

| Juego | Perfil | Motivo |
|-------|--------|--------|
| Valorant | 🏆 Esports | Necesita 240+ FPS |
| CS2 | 🏆 Esports | Competencia, baja latencia |
